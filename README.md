🗒️ Task Manager (H2 + JavaFX)
Небольшое десктопное приложение для управления задачами с дедлайнами, приоритетами и статусами выполнения.
 
 
 
⚙️ Возможности
✅ Добавление, редактирование и удаление задач
🕒 Указание даты и времени дедлайна
🔼 Установка приоритета (Low / Medium / High)
☑️ Отметка выполненных задач
💾 Сохранение всех данных в встроенную базу H2
🌙 Современный тёмный интерфейс на JavaFX
🖼️ Интерфейс
🧩 Технологии



Java 17+
JavaFX — интерфейс
H2 Database — хранение данных
Maven — управление зависимостями



🚀 Запуск проекта
Клонируй репозиторий:
git clone https://github.com/username/MyTasks-app.git



Перейди в папку проекта:
cd TaskManager

Собери и запусти через Maven:
mvn clean javafx:run



🗂️ Структура проекта
src/
 ├─ main/src/
 │   ├─ java/        → код приложения
 │   └─ resources/   → fxml, стили, конфиги
 └─ test/            → тесты



📌 Примечание
Приложение использует встроенную H2 базу, которая сохраняется локально в папке проекта — никакая установка БД не требуется.
