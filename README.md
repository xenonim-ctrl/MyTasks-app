🗒️ Task Manager (H2 + JavaFX)<br>
Небольшое десктопное приложение для управления задачами с дедлайнами, приоритетами и статусами выполнения.<br><br><br><br>


 
⚙️ Возможности<br>
✅ Добавление, редактирование и удаление задач<br>
🕒 Указание даты и времени дедлайна<br>
🔼 Установка приоритета (Low / Medium / High)<br>
☑️ Отметка выполненных задач<br>
💾 Сохранение всех данных в встроенную базу H2<br>
🌙 Современный тёмный интерфейс на JavaFX<br>
🖼️ Интерфейс<br>
🧩 Технологии<br><br><br><br>



Java 17+<br>
JavaFX — интерфейс<br>
H2 Database — хранение данных<br>
Maven — управление зависимостями<br><br>



🚀 Запуск проекта<br>
Клонируй репозиторий:
git clone https://github.com/xenonim-ctrl/MyTasks-app.git<br><br>



Перейди в папку проекта:<br>
cd TaskManager<br><br>

Собери и запусти через Maven:<br>
mvn clean javafx:run<br><br><br><br>



🗂️ Структура проекта<br>
src/
 ├─ main/src/
 │   ├─ java/        → код приложения
 │   └─ resources/   → fxml, стили, конфиги
 └─ test/            → тесты<br><br><br><br>



📌 Примечание<br>
Приложение использует встроенную H2 базу, которая сохраняется локально в папке проекта — никакая установка БД не требуется.
